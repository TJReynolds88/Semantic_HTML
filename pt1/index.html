HTML 5

The language for building web pages

New HTML5 Semantic Elements

  • <NAV>
  • <HEADER>
  • <ARTICLE>
  • <SECTION>
  • <ASIDE>
  • <FOOTER>

<NAV>

a set of navigation links.

ABOUT: The <NAV>element is intended only for major block of navigation links.

Browsers, such as screen readers for disabled users, can use this element to determine whether to omit the initial rendering of this content.

<HEADER>

A container for introductory content or a set of navigational links

Typically contains:

  • one or more heading elements (<H1> - <H6>)
  • logo or icon
  • authorship information
  • You can have several <Header> elements in one document

<ARTICLE>

Specifies independent, self-contained content

Potental Sources for <ARTICLE>:

  • Forum post
  • Blog post
  • News story
  • Comment

<SECTION>

A section in a document

Such as chapters, headers, footers, or any other sections of the document.

<ASIDE>

Defines some content aside from the content it is placed in

The aside content should be related to the surrounding content.

<FOOTER>

Defines a footer for a document or section.

Should Contain:

  • authorship information
  • copyright information
  • contact information
  • back to top links
  • related documents
